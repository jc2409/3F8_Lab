# 3F8 Inference Lab

Auxiliary code and data for the 3F8 Inference lab at the Cambridge Engineering Department.

## Contents

- `coursework.ipynb` — Jupyter notebook with the coursework template code
- `code.py` — Same template code as a standalone Python file
- `X.txt`, `y.txt` — Dataset files

## Getting Started

The easiest way to get started is to open the notebook directly in Google Colab:

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/cbl-cambridge/3F8-Lab/blob/main/coursework.ipynb)

The notebook includes setup instructions for downloading the data files.
